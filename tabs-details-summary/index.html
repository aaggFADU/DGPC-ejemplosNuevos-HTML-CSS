Tabs con elementos HTML <details> y CSS puro

* Para ver animación se requiere navegador Chrome 131 o posterior ó con la opción "Funciones experimentales de la plataforma web" habilitada.

¿Animar details?

Solía ser imposible, pero ahora, gracias al CSS moderno, ¡es bastante fácil de hacer!

¿Cómo se hace?

Hay formas diferentes de hacerlo. Una de ellas sería usar calc-size(), que tiene algunas cosas extra que se puede hacer con ella, pero actualmente la forma más fácil es usar una combinación de interpolate-size: allow-keywords; y transition-behavior: allow-discrete.

¿Compatibilidad?

La compatibilidad con navegadores no es buena pero mejorará. Además, es una gran mejora progresiva: no romperá nada para los usuarios de navegadores antiguos, solo no tendrán la animación que los usuarios de navegadores más nuevos tendrán.

Cambio a "tabs" propio, usando display: grid; y display: contents;.
