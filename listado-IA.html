Chats y Asistentes

ChatGPT
El asistente de IA más popular de OpenAI. Ofrece conversaciones naturales, responde preguntas, ayuda con redacción y programación, y permite integración con plugins de terceros.
Claude
Asistente de IA de Anthropic. Destaca por su capacidad de procesar grandes cantidades de texto, razonamiento ético y respuestas matizadas con menor tendencia a alucinaciones.
Deep Seek
Asistente conversacional especializado. Enfocado en tareas técnicas y de programación, con capacidad para resolver problemas complejos y generar código optimizado.
Qwen
Chat (web hasta dic. 2024). Modelo de lenguaje multimodal de Alibaba, especializado en tareas técnicas y creativas, con capacidad de procesamiento de texto, código e imágenes.
Gemini
El modelo de lenguaje más avanzado de Google. Multimodal, capaz de procesar texto, imágenes y código simultáneamente, con versiones Pro y Ultra para diferentes necesidades.
Manus AI
Avanzado agente IA general desarrollado por una empresa tecnológica china llamada Monica. Se destaca por su capacidad para abordar y ejecutar tareas complejas de manera autónoma, desde el análisis de datos hasta la resolución creativa de problemas, sin requerir intervención humana constante.
Mistral AI
Chat basado en el modelo Mistral. Destaca por su eficiencia y precisión en tareas de programación y análisis, con capacidad multilingüe y comprensión contextual avanzada.
Hugging Chat
Chat de Hugging Face. Plataforma de código abierto que ofrece acceso a múltiples modelos de lenguaje, con enfoque en transparencia y personalización.
Poe
Plataforma con múltiples modelos de IA. Permite acceder a ChatGPT, Claude, Llama y otros modelos desde una única interfaz, con posibilidad de crear bots personalizados y compartirlos.
Morphic
Asistente conversacional. Especializado en diseño y desarrollo, con capacidades de generación de código y resolución de problemas técnicos en tiempo real.
Storm
Asistente de Stanford.edu. Modelo de lenguaje desarrollado por investigadores de Stanford, especializado en razonamiento científico y académico con fuentes verificables.
Kimi
Asistente de IA chino con capacidades avanzadas de programación y comprensión de código, optimizado para desarrolladores. Código hasta 09/2023.
Allen AI
Entorno experimental del Allen Institute for AI para probar modelos de lenguaje en diversas tareas de razonamiento y comprensión. Playground (hasta 2023)
Glif
Mejorador de prompts (para flux). Herramienta especializada en optimizar instrucciones para modelos de generación de imágenes Flux, mejorando resultados visuales.
FLUX-Prompt-Generator
Generador de prompts en Huggingface. Aplicación que ayuda a crear instrucciones efectivas para el modelo Flux, con sugerencias de palabras clave y estructuras.
Prompt Hackers
Plataforma que ofrece técnicas y herramientas para optimizar instrucciones para diversos modelos de IA, con ejemplos y plantillas.
Runway Frames Prompt Builder
Constructor de prompts para Runway en ChatGPT. GPT especializado en crear instrucciones optimizadas para la generación de videos en Runway, con sugerencias de composición y estilo.

Prompter de Minimax en ChatGPT. GPT diseñado para crear instrucciones efectivas para la plataforma de video Minimax, optimizando resultados cinematográficos.

Generador de prompts a partir de imágenes. Aplicación que utiliza IA para crear instrucciones precisas para la reproducción de una imagen ya generada o de cualquier otro tipo de imagen.

Gestión de claves API de OpenAI. Portal para crear, administrar y monitorizar el uso de claves API para integrar servicios de OpenAI en aplicaciones propias.

Plataforma para probar y comparar diferentes modelos de IA. Permite evaluar el rendimiento de diversos modelos de lenguaje en las mismas tareas para identificar fortalezas y debilidades.
