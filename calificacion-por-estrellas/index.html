Calificación con estrellas con sólo HTML y CSS

Input con mask

La calificación se establece con un elemento <input type="range"> con atributos min="1", max="5" y valor inicial value="3".

El valor del atributo max se usa para determinar el número de estrellas.

La apariencia del elemento se obtiene usando la propiedad CSS mask y con las nuevas características de la función attr() de CSS (A inicios de febrero de 2025 los asistentes IA de VS Code marcan el código CSS como errado).

Tomado de Codepen de Temani Afif

A febrero de 2025 ChatGPT indica:
El error se debe a que el uso de attr() combinado con calc() y una conversión de tipo es parte de una especificación en desarrollo y aún no está adoptada de forma general por los motores de validación y los editores de código.

Para evitar problemas hay un "respaldo" con @supports not (d:attr(d type(*))) { } para que funcione en todos los navegadores que no lean las nuevas características de CSS.
