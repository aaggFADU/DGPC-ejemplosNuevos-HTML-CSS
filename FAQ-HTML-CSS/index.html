FAQ con elementos HTML <details> y CSS puro

* Para ver animación se requiere navegador Chrome 131 o posterior ó con la opción "Funciones experimentales de la plataforma web" habilitada.

🤔

Preguntas frecuentes

¿Se pueden animar los elementos details/summary?

Solía ser imposible, pero ahora, gracias al CSS moderno, ¡es bastante fácil de hacer!

¿Cómo se hace?

Hay formas diferentes de hacerlo. Una de ellas es usar calc-size(), que tiene algunas cosas extra que puedes hacer con ella, pero actualmente la forma más fácil es usar una combinación de interpolate-size: allow-keywords; y transition-behavior: allow-discrete.

¿Qué pasa con la compatibilidad con navegadores?

La compatibilidad con navegadores no es buena aún, pero mejorará. Además, es una gran mejora progresiva. Esto significa que no romperá nada para los usuarios de navegadores antiguos, que no tendrán la animación, mientras que los usuarios de navegadores más nuevos la tendrán.

¿Cuál es el truco?

¡No hay ninguno! ¡Es fácil de hacer y funciona muy bien!
