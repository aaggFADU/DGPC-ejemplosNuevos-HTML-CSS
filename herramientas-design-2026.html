Introducción

Antes de ver las herramientas

¿Qué es uso profesional?

Proyectos con entregables reales, clientes o productos propios. Se valora velocidad, calidad de output y continuidad sin interrupciones por límites de créditos.

¿Qué es uso académico?

Clases, ejercicios, prototipos de estudio. Prioridad: cero costo, instalación sencilla, curva de aprendizaje clara y sin necesidad de tarjeta de crédito.

Freemium ≠ Gratuito real

La mayoría de estas herramientas tienen planes "free" que funcionan como demos limitadas. Para uso académico serio hay que entender exactamente qué se puede hacer antes de llegar al paywall. Para uso profesional, el costo de suscripción suele recuperarse en horas.

Marco conceptual

No todas estas herramientas hacen lo mismo

Antes de evaluar cada herramienta por separado, conviene distinguir las cuatro categorías funcionales del ecosistema. Confundirlas lleva a comparar herramientas que no compiten entre sí.

◈ Canvas de diseño

Lienzo interactivo donde el diseñador trabaja directamente. Produce archivos de diseño versionables, no código listo para producción.

Pencil.dev Paper.design Figma Penpot

◈ Generación asistida

Toman un prompt o diseño existente y generan pantallas, variantes o código exportable. El output es el punto de partida, no el destino.

Magic Patterns MagicPath.ai Google Stitch

◈ Marketing / Landing

Orientadas a publicar páginas completas con rapidez. El foco está en el deploy, el SEO y la conversión, no en el sistema de diseño.

Loki.build

◈ Infraestructura de feedback

Se montan sobre una app existente para mejorar la comunicación diseñador↔agente IA. No generan diseño: mejoran la precisión del prompt.

Agentation

¿Por qué importa la distinción? Porque los criterios de evaluación difieren: un canvas se juzga por su fidelidad y versionado; un generador, por la calidad y exportabilidad de su output; una herramienta de marketing, por el tiempo hasta el deploy; y la infraestructura de feedback, por la reducción de ambigüedad en las instrucciones al agente.

Herramienta 01

Pencil.dev

Canvas de diseño en el IDE · MCP · Git
Gratis en Early Access Requiere agente MCP

Plataformas: Mac, Linux, Windows (soporte disponible, experiencia más estable en Mac/Linux). Extensión para VS Code y Cursor. App de escritorio standalone.

● Uso profesional

  • Diseño versionado en Git junto al código
  • Importar frames de Figma conservando vectores
  • Agentes MCP para generar pantallas en paralelo (Claude Code, Cursor, Antigravity, etc.)
  • Conexión a APIs reales y Playwright para testing
  • Costo real: el canvas es gratis Early Access; el motor IA depende del agente. Con Antigravity u otro agente MCP propio: sin costo adicional

● Uso académico

  • Demostrar integración diseño-código en clase
  • Ejercicio de branching de diseño como código
  • Alternativa liviana a Figma dentro del IDE
  • Barrera real: requiere un agente MCP configurado — Claude Code ($20/mes) es el más documentado, pero Antigravity u otros agentes MCP funcionan igualmente
  • Sin ningún agente IA: funciona como canvas manual, sin generación automática

Veredicto: El canvas es gratis. El motor IA lo aporta el agente MCP conectado: Claude Code ($20/mes) es el camino más documentado, pero cualquier agente con soporte MCP funciona — incluyendo Antigravity dentro de su cuota propia. Para estudiantes sin ningún agente IA, el canvas manual sigue siendo útil para branching de diseño y versionado en Git.

Herramienta 02

Loki.build

Landing pages · Builder nativo IA · Web
Freemium Marca de agua en free Marketing / Landing-first

Generación de páginas de aterrizaje completas desde un prompt. No es una herramienta de diseño de producto: el foco está en publicar rápido con buen SEO y conversión, no en construir sistemas de diseño. No requiere instalación — 100% web.

● Uso profesional

  • Prototipo rápido de propuesta para cliente
  • Test A/B visual con función Remix
  • SEO automático + responsive por defecto
  • Publicación con dominio propio solo en Premium
  • Sin marca de agua solo en Premium
  • Gratis útil hasta: mockups para aprobar concepto, no para entrega final

● Uso académico

  • Ejercicio: generar landing desde un brief textual
  • Comparar output IA vs diseño manual
  • Análisis crítico de jerarquía visual generada
  • No requiere cuenta para generación básica
  • Límite: publicación tiene marca de agua, dominio propio requiere pago
  • Ideal para clase: el proceso importa más que el deploy

Veredicto: El mejor de la lista para uso académico inmediato. Sin cuenta, sin instalación, sin costo para generar y revisar. La limitación del dominio propio no afecta al aula.

Herramienta 03

Agentation

Anotación visual del DOM · Open Source · React
100% Gratuito Open Source Infraestructura · No canvas

Biblioteca React que se monta sobre cualquier app en desarrollo para anotar elementos del DOM y generar contexto estructurado para agentes IA. Cero dependencias adicionales (solo React 18+).

● Uso profesional

  • Eliminar ambigüedad en el feedback de UI al agente
  • Anotar estados de animación congelados para revisión
  • Generar contexto Markdown preciso con selector CSS + ruta DOM
  • Compatible con Claude Code, Cursor, Windsurf
  • Sin límites: open source, sin créditos, sin plan

● Uso académico

  • Enseñar estructura del DOM de forma visual e interactiva
  • Ejercicio: identificar selectores CSS en apps reales
  • Demostrar jerarquía de componentes React en clase
  • Instalación: npm install agentation -D
  • Sin límites. Requiere entorno React 18+, no funciona en móvil para edición

Veredicto: La única herramienta de la lista completamente libre de restricciones. Ideal como infraestructura invisible en proyectos académicos y profesionales. Muy útil para enseñar DOM y selectores CSS.

Herramienta 04

Magic Patterns

Prototipado IA · Sistemas de diseño · React/Vue/Tailwind
20 generaciones gratis Créditos se agotan Export · No MCP-native

● Uso profesional

  • Iteración sobre sistemas de diseño existentes
  • Exporta React, Vue y Tailwind CSS — integración vía export, no agente MCP
  • Captura de pantalla desde cualquier sitio con extensión Chrome
  • Plan Hobby $15–19/mes: 80–100 msg + rollover 2 meses
  • Gratis útil para: exploración inicial, proof of concept
  • Ediciones manuales y exports no consumen créditos

● Uso académico

  • 20 generaciones gratis son suficientes para 1–2 ejercicios completos
  • Cada alumno puede crear cuenta gratuita
  • Ejercicio: importar sistema de diseño real y extenderlo
  • Canvas colaborativo en tiempo real
  • Límite crítico: agotadas las 20 generaciones, requiere pago
  • Estrategia: usar gratis para concepto, refinar manualmente

Veredicto: Las 20 generaciones gratuitas son útiles pero se agotan rápido. En aula funciona bien si el ejercicio está bien acotado. Para profesionales, el costo del Hobby se justifica desde el primer proyecto.

Herramienta 05

Paper.design

Canvas nativo HTML/CSS · Agentes MCP · Alta fidelidad
100 llamadas MCP/semana gratis Restrictivo en free

Canvas construido íntegramente sobre HTML y CSS. Lo que se ve es lo que renderiza el navegador. Soporta gamut P3, shaders animados y conexión directa a APIs.

● Uso profesional

  • 100 llamadas MCP/semana gratis: insuficiente para flujo activo
  • Plan Pro $20/mes: 1M llamadas/semana — uso real
  • Diseños conectados a APIs y spreadsheets con datos reales
  • Exportación de video solo Pro
  • Sincronización bidireccional código ↔ lienzo

● Uso académico

  • 100 llamadas/semana alcanzan para ejercicios puntuales
  • Excelente para demostrar que HTML/CSS ES el diseño
  • Mostrar filtros CSS y shaders directamente en diseño
  • 200MB por archivo en plan free
  • Límite: el ritmo de agente se corta rápido en proyectos largos
  • Uso manual (sin IA) no tiene límites en free

Veredicto: En free, Paper es útil como demostración y para ejercicios breves. El argumento conceptual (HTML/CSS como lienzo) es poderoso para enseñanza. Para profesionales, el Pro a $20/mes es obligatorio si se usan agentes.

Herramienta 06

MagicPath.ai

Diseño IA empresarial · Liquid Glass · Valoración $100M
20 créditos diarios gratis Renovación diaria

20 créditos que se renuevan cada día (no por mes) hacen que el plan free sea excepcionalmente generoso comparado con competidores.

● Uso profesional

  • 20 créditos/día renovables = acceso sostenido para exploración
  • 5 importaciones de Figma/mes en free
  • Plan Creator $20/mes: 400 créditos/mes + 25 imports Figma
  • Descarga de código fuente completo solo Pro $30/mes
  • Sincronización directa con Cursor u otros IDEs
  • Generación de variantes instantáneas desde un prompt

● Uso académico

  • El más generoso en free por la renovación diaria
  • Cada sesión de clase puede usar créditos frescos
  • Ejercicio: generar 3 variantes de una pantalla y comparar
  • Interfaz tipo "Liquid Glass" — útil para discutir tendencias estéticas
  • Límite: sin descarga de código fuente en free
  • Ideal para enseñar workflow de variantes rápidas

Veredicto: El plan gratuito más sostenible de la lista para uso académico continuo. La renovación diaria lo hace viable clase a clase. Para profesionales que necesitan exportar código, el plan Creator es el mínimo.

Emergente · A observar en 2026

Penpot MCP

Design → Code con agentes · Open Source · Sin lock-in
Emergente 2026 Open Source No mainstream aún

Penpot — ya listado como canvas de diseño — está desarrollando una capa de Model Context Protocol que expone el archivo de diseño directamente a agentes IA. Permite el flujo design → code con agentes sin depender de plataformas propietarias ni créditos de terceros.

● Por qué es estratégica

  • Flujo design → code via agente completamente open source
  • Sin lock-in: ni Figma, ni Claude Code, ni créditos de terceros
  • Compatible con cualquier agente que soporte MCP (Claude, Cursor, etc.)
  • Penpot es self-hosteable — flujo 100% soberano posible
  • Hoy: experimental / en desarrollo activo no production-ready

● Potencial académico

  • Enseñar MCP como protocolo — no como vendor feature
  • Ejercicio: conectar un agente a un archivo Penpot real
  • Demostrar ecosistema open source alternativo al stack propietario
  • Sin costo, sin cuenta paga, sin límites de créditos
  • Barrera: requiere configuración técnica, no plug-and-play aún

Veredicto: No reemplaza a ninguna herramienta de la lista hoy, pero representa la dirección hacia un ecosistema design → code abierto. Estratégica para quien enseña arquitectura de herramientas IA, no solo su uso superficial. Monitorear durante 2026.

Conclusión

Stack recomendado sin costo

● Stack profesional free

  • Figma free — 3 archivos para el trabajo activo
  • MagicPath.ai — 20 créditos/día para iteración visual
  • Loki.build — mockups de landing para cliente
  • Agentation — feedback preciso de UI al agente
  • Bolt.new — full-stack para prototipos rápidos

● Stack académico free

  • Penpot — base de diseño sin límites (o Figma Education)
  • Loki.build — generación IA sin cuenta, sin costo
  • MagicPath.ai — créditos diarios sostenibles para cada clase
  • Agentation — enseñar DOM y selectores de forma visual
  • Bolt.new — full-stack sin instalación de entorno

⚠ Herramientas con condiciones a verificar

Pencil.dev — el canvas es gratis. El motor IA lo aporta el agente MCP conectado: Claude Code ($20/mes) es el camino más documentado, pero cualquier IDE con soporte MCP (Antigravity, Cursor, Windsurf, etc.) funciona con la cuota propia del agente — sin costo adicional.

UX Pilot / Magic Patterns — los límites gratuitos se agotan en pocas sesiones.

Paper.design — 100 llamadas MCP/semana son insuficientes para trabajo activo con agentes.
